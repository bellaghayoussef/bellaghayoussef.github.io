Project Overview

Tasareeh App is a mobile application for managing digital permits, making it easy for users to apply, track, and store permits securely on their devices.

  • Mobile-first, intuitive interface
  • Easy permit application and approval
  • Push notifications for status updates
  • Secure document storage
  • Multi-language support
